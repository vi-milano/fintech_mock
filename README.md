# Fintech-mock

Fintech-mock is a test project, ReactJS powered, mocked with GraphQL.
[fintech-mock] - Click here for the online demo!

# How to run locally?

- Clone the repository
- Run the following:

```sh
$ cd fintech_mock
$ yarn start
```

- Then, run `localhost:3000` from your browser
- Have fun!
